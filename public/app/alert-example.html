Componente de Alertas

Sobre o Componente

Este componente de alertas foi criado para fornecer uma maneira consistente e fácil de exibir mensagens para os usuários em toda a aplicação. Ele suporta diferentes tipos de alertas, como sucesso, erro, aviso e informação.

Como Usar

Para usar o componente de alertas em suas páginas, siga estas etapas:

1. Inclua os arquivos CSS e JavaScript necessários:

<link rel="stylesheet" href="../css/alert.css"> <script src="../js/alert.js"></script>

2. Use o objeto global Alert para exibir alertas:

// Alerta de sucesso Alert.success('Título', 'Mensagem'); // Alerta de erro Alert.error('Título', 'Mensagem'); // Alerta de aviso Alert.warning('Título', 'Mensagem'); // Alerta de informação Alert.info('Título', 'Mensagem'); // Alerta de confirmação Alert.confirm( 'Título', 'Mensagem', () => { /* código quando confirmado */ }, () => { /* código quando cancelado */ } );

3. Personalize os alertas com opções adicionais:

Alert.success('Título', 'Mensagem', { icon: '../caminho/para/icone.png', buttons: [ { text: 'Texto do Botão', class: 'primary', onClick: () => { /* código */ } } ], closeOnOverlayClick: false });
